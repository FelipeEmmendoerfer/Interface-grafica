# Interface gráfica de login

Este projeto é uma aplicação simples em Python para uma tela de login com interface gráfica usando PySimpleGUI.

## Funcionalidades

- Tela de login com nome e senha
- Opção para mostrar a senha
- Opção para salvar o login
- Persistência dos dados em um arquivo JSON

## Como executar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute o programa:
   ```bash
   python cadastro.py
   ```

## Requisitos

- Python 3.8+
- PySimpleGUI
